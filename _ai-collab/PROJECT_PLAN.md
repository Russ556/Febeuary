## 2. MVP 범위

### 2.1 포함 (In Scope)

| # | 기능 | 우선순위 | 설명 |
|---|------|----------|------|
| F1 | `메인 화면 (Main)` | P0 (필수) | `중앙 이름/직함 표시, 4방향 네비게이션 가이드 표시` |
| F2 | `키보드 네비게이션` | P0 (필수) | `방향키(↑↓←→) 입력 시 해당 섹션으로 부드럽게 이동` |
| F3 | `섹션별 컨텐츠` | P0 (필수) | `PROFILE(좌), CONTACT(우), EDUCATION(상), PROJECTS(하) 데이터 렌더링` |
| F4 | `반응형 모바일 지원` | P1 (중요) | `모바일에서는 스와이프 또는 터치 버튼으로 이동 지원` |

### 2.2 제외 (Out of Scope — v2 이후)

- `CMS 연동 (초기에는 로컬 JSON 데이터 사용)`
- `다크 모드 (초기에는 화이트 테마 집중)`
